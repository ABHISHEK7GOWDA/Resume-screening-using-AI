# TalentRank AI Frontend

Run the backend and frontend in two terminals.

Do not use the VS Code Live Server "Go Live" button for this project. This is a Vite React app, so it must run through Vite.

## Backend

```bash
cd ../backend
npm install
npm start
```

The API runs at `http://localhost:3000`.

## Frontend

```bash
cd ../frontend
npm install
npm start
```

This opens the app automatically. If it does not, open `http://127.0.0.1:5173`.
